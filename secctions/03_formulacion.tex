\section{Formulación del problema}

\subsection{Problema general}

¿Cómo modelar el crecimiento del acceso a internet mediante ecuaciones diferenciales, usando datos del INEI entre 2014 y 2024?

\subsection{Problemas específicos}

\begin{itemize}
    \item ¿Qué modelo representa mejor el crecimiento nacional del acceso a internet: el modelo exponencial o el modelo logístico?
    \item ¿Cómo se comporta el acceso a internet en los hogares urbanos y rurales durante el periodo 2014-2024?
\end{itemize}
